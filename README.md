## Supported Platforms

Nstbrowser runs on Windows, macOS (both Intel and Apple Silicon), and Linux. The desktop app works as a local agent that connects with the cloud dashboard, which handles profile sync and team management.

The automation SDK is compatible with Puppeteer, Playwright, and Selenium, covering essentially every mainstream browser automation stack.
